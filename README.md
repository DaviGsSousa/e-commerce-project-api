## Description

That's my first Nest project, so far, the goal is to be used only for studying Nest, it's a e-commerce Api. At this project i'm using Nest and prisma with a postgre database

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
